# Query options 
-X GET http://dressapi/api/[TABLE|MODULE]/[IDs/][filters]/[wr]/[p/pag,elements_per_page]/[ob/id|id-ASC|id-DESC]

[ob|order-by] order of the table (the next value must be the name of the item like "id-DESC")
[wr|with-relations] change the index value with a related value of foreign table
[p|page] with explicit impagination (the next value must be the number of page and optionally the max row per page) 

[filters] you can you use more than one conditions with this operators:
[=]
[<=]
[=>]
[<]
[>]
[~] contains the text

for example "/api/page/id_user>=100/title~Welcome"
